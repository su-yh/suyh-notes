wget -O install.sh http://download.bt.cn/install/install-ubuntu_6.0.sh && sudo bash install.sh

```

sudo apt-get update
```



### 查看ubuntu 系统版本信息

```shell
lsb_release -a
```

示例：

```txt
user@user:~$ lsb_release -a
No LSB modules are available.
Distributor ID:	Ubuntu
Description:	Ubuntu 24.04 LTS
Release:	24.04
Codename:	noble
```
