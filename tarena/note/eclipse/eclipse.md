## eclipse 的一些使用及技巧

### eclipse 中的一些常用快捷键
- `alt + /`
	> API 类等的提示
- `ctrl + /`
	> 单行注释与取消
- `ctrl + shift + /`
	> 多行注释
- `ctrl + shift + \`
	> 取消多行注释
- `ctrl + shift + t`
	> 快速查找类对象
- `ctrl + t `
	> 查看类的详细信息，然后可以搜索这个类的所有属性和方法<br>
	> 查看类的方法及属性<br>
	> 查看此类的子类及父类<br>
	> 选中一个类然后使用 `ctrl + t `<br>
- `ctrl+o`
	> 查看类中的方法，假如想看类中父类的方法也可以继续`ctrl+o`
- `ctrl+n`
	> 调出新建窗口
- `ctrl+shift+o`
	> 快速导入包
